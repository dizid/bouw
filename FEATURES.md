# Breijer Klussen App - Features

## Klussen (Job Entry)
- Select worker + house number
- 5 task types with minutes tracking:
  - Binnen opruimen
  - Buiten balkon opruimen
  - Zonnescherm verwijderen (+ terugplaatsen/afstandverklaring options)
  - Glasbreuk balkonafscheiding (+ aantal)
  - Diversen (+ bewoner contact info)
- Notes (opmerkingen) per task
- Photo capture for documentation
- Remembers last selected worker

## Dashboard (/dashboard)
- **Medewerkers tab**: Add/remove workers, click name to filter
- **Overzicht tab**: Stats (houses, hours), recent sessions, CSV export, print
- **Per Woning tab**: House-by-house breakdown with photos

## Technical
- Soft delete for workers (preserves history)
- Filter by worker with stats update
- Mobile-first design for construction workers
