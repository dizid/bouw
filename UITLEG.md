# Breijer Klussen App - Uitleg

## Voor Medewerkers

### Klus Registreren
1. **Selecteer jezelf** - Dropdown bovenaan (onthoudt je keuze)
2. **Vul huisnummer in**
3. **Maak foto's** - Optioneel, van de situatie
4. **Vink taken aan** die je hebt gedaan:
   - Binnen opruimen
   - Buiten balkon opruimen
   - Zonnescherm verwijderen (+ terugplaatsen checkbox)
   - Glasbreuk balkonafscheiding (+ aantal)
   - Diversen (+ bewoner naam/telefoon)
5. **Vul minuten in** per taak
6. **Opmerkingen** indien nodig
7. **Opslaan** - Groene knop onderaan

---

## Voor Managers

### Dashboard → /dashboard

**Tab: Medewerkers**
- Nieuwe medewerker toevoegen
- Verwijderen met × knop
- Klik op naam = filter overzicht op die persoon

**Tab: Overzicht**
- Totaal woningen en uren
- Stats per medewerker
- Recente sessies (laatste 20)
- **CSV** knop = download spreadsheet
- **Print** knop = afdrukken

**Tab: Per Woning**
- Tabel met alle woningen + minuten per taak
- Klik op rij = details + foto's bekijken

**Tab: Fases**
- Dropdown om fase te selecteren (1, 2, 3...)
- Fase = eerste 50 afgeronde woningen op volgorde van afronding
- Tabel met woningen in die fase + totaal uren
- **PDF Rapport** knop = download PDF met:
  - Samenvatting per woning
  - Taken gedaan (checkmarks)
  - Totaal uren
  - Foto's

### Filteren
- Klik medewerker naam → overzicht toont alleen die persoon
- **× Wis filter** knop om terug te gaan naar alles
